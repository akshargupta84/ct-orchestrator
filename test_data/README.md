# Sample Test Inputs for CT Orchestrator

This folder contains sample data you can use to test the CT Orchestrator tool.

## Files Included

1. **sample_campaigns.json** - Pre-configured campaign scenarios
2. **sample_creative_trix.csv** - Creative matrix with channel/impression data
3. **sample_results_passed.csv** - Test results where most creatives pass
4. **sample_results_failed.csv** - Test results where most creatives fail
5. **sample_results_mixed.csv** - Realistic mixed results
6. **sample_hypotheses.txt** - Example testing hypotheses

## Quick Start

### Testing the CT Planner

Use these values in the Streamlit form:

**Scenario 1: Small Campaign**
- Campaign Name: Q1 Brand Awareness Push
- Brand: FreshBite Snacks
- Budget: $3,000,000
- Videos: 2
- Display: 4
- Primary KPI: Awareness

**Scenario 2: Medium Campaign**
- Campaign Name: Summer Product Launch
- Brand: TechFlow Electronics  
- Budget: $15,000,000
- Videos: 6
- Display: 12
- Primary KPI: Consideration

**Scenario 3: Large Campaign**
- Campaign Name: Holiday Mega Campaign
- Brand: LuxeStyle Fashion
- Budget: $75,000,000
- Videos: 12
- Display: 25
- Primary KPI: Purchase Intent
